INFORMATION

Greenfoot is an IDE(Integrated Development Enviroment) that uses Java to teach students about software skills like creating games. Unlike MakeCode, in order to compete in this competition, you would need to download this program into your computer. Greenfoot allows for anyone to have acess to an easy to use software where they can test their creativity and critical thinking when it comes to creating a game. On their website, you can find many games that other creators published!
